# Общие переменные для шаблонов

[English](README.md)

Позволяет быстро использовать в любом .twig витрины контактные данные магазина, сведения о покупателе или даже GET-параметры запроса. Не нужно снова и снова добавлять одни и те же правки в каждом контроллере.

---

## Возможности

Модификация подключается в `system/engine/loader.php` и дополняет массив `$data` перед рендером шаблона. Работает **только на витрине** (не в админке).

### `_config` — настройки магазина

- `{{ _config.name }}` — название магазина
- `{{ _config.address }}` — адрес
- `{{ _config.geocode }}` — геокод
- `{{ _config.email }}` — e-mail
- `{{ _config.telephone }}` — телефон
- `{{ _config.fax }}` — факс
- `{{ _config.open }}` — часы работы
- `{{ _config.url }}` — URL магазина

### `_customer` — данные текущего покупателя

- `{{ _customer.id }}` — ID покупателя (0 для гостя)
- `{{ _customer.firstname }}` — имя
- `{{ _customer.lastname }}` — фамилия
- `{{ _customer.email }}` — e-mail
- `{{ _customer.telephone }}` — телефон
- `{{ _customer.newsletter }}` — подписка на рассылку
- `{{ _customer.group_id }}` — ID группы покупателей

### `_get` — GET-параметры запроса

- `{{ _get.route }}` — текущий route и т.п.

## Другие модули

Для подстановки аналогичных переменных в HTML-контенте, редактируемом в админке, используйте модуль **ShortCodes** — см. профиль автора на OpenCart Marketplace или репозиторий shortcodes в GitHub.
Модули можно использовать вместе или по отдельности.

## Установка

1. Загрузите .ocmod.zip через **Расширения → Установка**.
2. Перейдите в **Расширения → Модификаторы** и нажмите **Обновить**.
3. При необходимости очистите кэш модификаций и шаблонов.
